
Cloudeebus
==========

Cloudeebus - DBus for the Cloud - is a component which enables calling DBus
 methods and registering on DBus signals from Javascript.


Install
-------

    npm install node-dbus
    npm install xmldom


Examples
--------

    node ./test/test-proxy.js
